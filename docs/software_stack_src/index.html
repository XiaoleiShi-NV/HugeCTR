NVIDIA Driver

Release v3.0 is based on CUDA 11.0.221, which requires NVIDIA driver release 450.51.

However, if you are running on Tesla (for example, T4 or any other Tesla board), you may use NVIDIA driver release 418.xx or 440.30.

The CUDA driver's compatibility package only supports particular drivers. 1
